Setup Guide

How to deploy this dashboard globally, set up Supabase, and use every feature

🌍 Deploy to GitHub Pages (Free, Global)
Step 1 — Go to github.com → New repository → Name it padosipro-mcc
Step 2 — Upload this HTML file as index.html
Step 3 — Go to repo Settings → Pages → Source: main branch → Save
Step 4 — Your URL: https://yourusername.github.io/padosipro-mcc
Step 5 — Share this URL with all 4 team members

✅ Free, instant, globally accessible, always online
⚡ Supabase Real-Time Setup (Free)
Step 1 — Go to supabase.com → Sign up → New Project
Step 2 — SQL Editor → paste + run the schema from setup screen
Step 3 — Settings → API → copy Project URL + anon public key
Step 4 — Open dashboard → click your name → Enter Supabase URL + Key
Step 5 — All 4 team members see updates instantly ⚡
